# Scalar en Harbour
Clases que dan más funcionalidad a los diferentes tipos de datos de Harbour

Para añadirlas a cualquier programa, simplemente hay que copiar la carpeta \Scalar omitiendo los ficheros .test al programa que se deseen integrar y añadirlos dentro del .hbp
En este repositorio se incluye el sistema de test para comprobar que todos los métodos funcionan y están testeados. Para los tests hay que compilarlo con c.bat y ejecutar el test.exe. Se puede descargar hbunit independientemente en https://github.com/VictorCasajuanaMas/hbunit para poder probarla.
También se incluye el debug que puede ser ejecutado desde VsCode para realizar pruebas o añadir más métodos.

Cualquier persona puede aportar mejoras a los métodos existentes o crear métodos nuevos, pero deben ir acompañado de su test con hbunit para poder aceptar las modificaciones.
